KindledSpirits

Project Description

VR Coop Game with single player storyline and multiplayer implementation using Normcore; utilizing free assets and VR interactions with UI elements in 3D space.

Player can choose between singleplayer and multiplayer game modes. Include Tutorial levels for each game mode. In the Singleplayer Mode, the goal is to find the stolen key to be able to enter their cabin; fighting enemies with marshmallows and keeping themselves warm with campfires and torches. In the Multiplayer Mode, the goal is to survive the night by lighting all the campfires in the level above 75%.
